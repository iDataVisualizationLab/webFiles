    Schematic overview


    Leader algorithm


    Input: n scatterplots and a threshold r

    1. We initialize the leader list L= Ø
    2. For each scatterplot Si, we find the nearest leader in L (dist(Si,L)≤ r)
    3. If we find a nearest leader, we add Si to the follower list of the nearest leader
    4. Otherwise, we make Si a new leader and add Si into L

    The complexity is roughly O(n)



    J. Hartigan. Clustering Algorithms. John Wiley & Sons, New York, 1975
